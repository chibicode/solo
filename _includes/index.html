How To Use

  • Mark the big gray button of Softwares you need to add install.
  • Hit the Download button.
  • Keep both files in the same folder.
  • Open Terminal
  • Go to the folder where you saved the downloaded files
    • cd Downloads
  • Run the shell script as root user.
    • sudo sh cleanstart.graphics.sh

    Warnings :)

  • Check the 'Updated on' date on each software listed. That is the date that we last checked the PPA to see if its still being updated.
  • This is a collection of ppa's that we have collected since we started using Ubuntu our studio.
  • There might be compatibility issues on some of the latest versions like Scribus. So Google around and/or visit the PPA page on Launchpad and find out wheather it works for you.
  • There are many different Libre softwares that do the same tasks.. And there are many PPA's with different packages. This is based on just our preference.
